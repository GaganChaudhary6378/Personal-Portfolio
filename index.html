    • Metric Converter

    It's a simple metric converter that takes input unit from the user and converts one unit into several other units.

    It comes with predefined values, so when the user opens the page he can see the conversion for default one unit. This comes with three pre-defined units length(meter/feet), volume(litres/gallons) and mass(kilograms,pounds) to help you get started faster.

    BlackJack Game

    It is a random game which works by using Math.random function. What is actually does if the user click on Start Game then it generates any two random cards and display their sum and for BlackJack the sum must be equal to 24. So if it is 24 then user will win otherwise it asks the user whether he wants to draw a new card or not and again does the same thing. If the sum is greater than 24 then the user is out of the game.

    It has the following components:-

    • Start Game

    • New Card

    epeigon (An emergency health website)

    This website is mainly designed for those people who need instant ambulance service in case of emergeny.

    What it does it actually have a emergency button so when the user clicks on that button it doesn't ask for any login or payment etc. because our priority is to provide instant help for the needy people. So after clicking emergency button it sends the emergency signal to all the hospitals in 10km of radius. So instead of calling different different hospitals just press the one button and in minutes ambulance is at your doorstep.

    Auditor (Music Website)

    It has following components:-

    • Create Space

    • Create Playlist

    • Listen to the song of the day
